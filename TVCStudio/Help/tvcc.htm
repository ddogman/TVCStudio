TVCC

 

A TVCC program egy parancssoros fordító program a Videoton TV computerhez, mely képes a TV computerre írt Z80 assembly, vagy basic file-ok lefordítására. Kimenetként CAS és/vagy WAV filet állít elõ, melyek emulátorban, vagy az eredeti hardveren is használhatóak.

 

A program használata:

TVCC.exe -i file -t=Z80,bas [-a] [-bl] [-sa=999999] [-o=cas,wav,lst,obj]

A parancssori argumentumok jelentése:

-i

A bemeneti szöveg file útvonallal együtt. Ha a file elérési útvonalában szóköz is van (Program Files) akkor az útvonalat idézõjelbe kell tenni!

-t=Z80, bas

A bemeneti file típusa. Értékei: Z80 (assembly program), vagy bas(basic program)

-a

Az autostart flag-et az elkészült cas és wav file-okban igazra állítja. (Opcionális)

-bl

A megadott paraméterrel a fordító basic betöltõt generál az assembly programhoz(csak a -t=Z80 kapcsolóval együtt használatos), ha ez a paraméter nincs megadva, csak a lefordított gépi kód kerül CAS file-ba, melyet saját töltõvel kell betölteni.

-sa=999999

(Csak a -t=Z80 és -bl kapcsolókkal együtt). Ha több 'ORG' utasítást használtunk az assembly programunkban a kapcsoló használata kötelezõ. A kapcsolóval tudjuk megadni a program indítási címét (tizes számrendszerben) a basic loader számára. Az értéke 0-65535 között lehet.

-o=cas,wav,lst,obj

A kapcsoló használatával a kimeneti file adható meg. (lst:lista file, obj:nativ gépi kód, cas: cas file, wav: wav file). Ha a kapcsoló nincs megadva, akkor a cas és wav file-ok automatikusan elkészülnek.

 

Használati példák:

Basic program

TVCC.exe -i c:\programok\programom.bas -t=bas –a

 

Assembly program betöltõvel

TVCC.exe -i c:\programok\programom.asm -t=Z80 -a –bl

 

Assembly program betöltõvel a 12456-os címen

TVCC.exe -i c:\programok\programom.asm -t=Z80 -a -bl -sa=12456

 

Assembly program betöltõ nélkül

TVCC.exe -i c:\programok\programom.asm -t=Z80

 

Assembly lista file készítése

TVCC.exe -i c:\programok\programom.asm -t=Z80 –ol

 

Assembly cas file készítése

TVCC.exe -i c:\programok\programom.asm -t=Z80 -o=cas

 

 Assembly wav file készítése

TVCC.exe -i c:\programok\programom.asm -t=Z80 -o=wav

 

Assembly cas és wav file készítése

TVCC.exe -i c:\programok\programom.asm -t=Z80
