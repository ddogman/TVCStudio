TVCStudio

 A TVCStudio egy grafikus fejlesztõkörnyezet, mely támogatja a programfejlesztést Videoton TV Computerre Z80 assembly, valamint BASIC programozási nyelveken.

 


 

A program szekciói

 

Eddigi Programok szekció

A szekció a program tetszõleges részén dokkolható. A megnyitott, vagy újonnan létrehozott programok ebben a listában jelennek meg, mely a program bezárása után is megmarad.

 

A programmal kapcsolatban a következõ információk jelennek meg:

 

A program típusa. Ez jelenleg Assembly, vagy Basic lehet. Ha a típusra duplán kattintunk, akkor a program megnyílik szerkesztésre.

A program neve (ha a programot idõközben eltávolították, akkor a program neve piros színnel jelenik meg)

A könyvtár, amelyben a program található

Mûveleti ikonok

Automatikus indítás jelölõ négyzet. Ha be van pipálva, akkor a programban az Autostart flag igazra lesz állítva. (A program automatikusan elindul)

Írásvédett jelölõ négyzet. Ha be van pipálva, akkor a program írásvédettként lesz megjelölve.

WAV file létrehozása. Ha be van pipálva, akkor a fordítás során wav állomány is létrejön.

 

Wav állomány létrehozásakor három paramétert tudunk megadni:

Frekvencia eltolás százalékos értéke (alapértéke 0)

Block bevezetõ jel hosszúsága (alapértéke 4812)

Fejléc- és adatblokk közötti rés (gap) jeltávolsága (alapértéke 1000)

 

A három paraméter változtatásával turbó betöltõ funkciót valósíthatunk meg.

A következõ értékekkel a betöltés 50 százalékban gyorsítható:

Frekvencia eltolás : 20

Block bevezetõ jel hosszúság: 600

Fejléc, adatblock gap: 200

 

CAS file létrehozása. Ha be van pipálva, akkor a fordító CAS file-t is generál, mely emulátorban, vagy a TV computeren azonnal használható.

OBJ file létrehozása. Ha be van pipálva, akkor a fordító az elkészített natív tárgykódot OBJ file-ba menti. (Csak assembly program esetében elérhetõ)

Program beágyazása BASIC betöltõbe. Ha be van pipálva, akkor az assembly programhoz a fordító BASIC betöltõt generál, és azt menti a CAS, vagy WAV file-okba. (Csak assembly program esetén elérhetõ)

Lista file létrehozása. Ha be van pipálva, akkor az assembler lista file-t is készít. (Csak assembler program esetén)

A megadott beállítások a TVCStudio újraindítása után is megmaradnak.

A mûveleti ikonok jelentése balról jobbra a következõ:

Program fordítása

A gombra kattintva a program fordítása megkezdõdik. A mûvelettel kapcsolatos üzenetek a program alsó, Kimenet szekciójában jelennek meg.

Lefordított állományok törlése

A gombra kattintva törlõdnek a fordítás során keletkezett file-ok.

Program lejátszása (csak sikeres fordítás után jelenik meg)

A gombra kattintva elindul a wav file-okhoz beállított zenelejátszó program, amellyel lejátszhatjuk a létrehozott file-t a TV Computernek.

Generált BASIC loader megnyitása (csak sikeres fordítás után jelenik meg)

A gombra kattintva megtekinthetjük a generált BASIC betöltõ forráskódját. (Csak assembly programok esetén jelenik meg a gomb)

Generált lista file megtekintése (csak sikeres fordítás után jelenik meg)

A gombra kattintva megtekinthetjük a generált lista file-t. (Csak assembly programok esetén jelenik meg a gomb)

Program indítása emulátorban (csak CAS file generálása esetén jelenik meg) 

A gombra kattintva elindul az emulátor, mely argumentumként megkapja a létrehozott CAS file-t. Ha a beállításokban nem adtuk meg az emulátor útvonalát, akkor a gomb inaktív.

Program eltávolítása listából.

A program nem törlõdik fizikailag, csak a listából tûnik el. Ismételt megnyitáskor újra megjelenik a listában.

 

Kimenet szekció

A szekció a program tetszõleges részén dokkolható. Minden, a program által küldött üzenet ebben a szekcióban jelenik meg. A program bezárása után, vagy a felsõ eszköztáron a radír ikonra kattintva, az összes üzenet automatikusan törlõdik.

 

Kódszerkesztõ szekció

A szekcióban megnyitott dokumentumok a program tetszõleges részén dokkolhatóak. Minden szerkesztésre megnyitott program ebben a szekcióban nyílik meg szerkesztésre.

A kódszerkesztõ a következõ funkcionalitásokat támogatja:

-          Alap szerkesztési funkciók (kivágás, másolás, beillesztés, törlés)

-          Szintaxis kiemelés: a kulcsszavak mind BASIC, mind pedig assembly program esetében külön színnel jelennek meg

-          Kódkiegészítés: gépeléskor automatikusan megjelenik a kódkiegészítõ ablak, melybõl kiválaszthatjuk a megfelelõ utasítást. Az ablak a CTRL+Space billentyû kombinációval is bármikor elõhívható.

-          Sorok számának megjelenítése, elrejtése: Az eszköztáron a # gombbal szabályozható.

-          Kód összecsukás (csak assembly programok esetében): Lehetõség van assembly programok gépelésekor összecsukható kód szekciókat létrehozni, melyek nagyban megkönnyíthetik a hosszú programok átláthatóságát. A kódszekció a ;{ karakterekkel kezdõdik és a }; karakterek zárják.

-    Szimbólumtábla (csak assembly programok esetében): Ebben a listában jelennek meg az assembly programon belül létrehozott szimbólumok, azok értékei, és a definiáló sor száma.

-    Programhibák (csak assembly programok esetében): Ebben a listában jelennek meg az esetleges programhibák.

Példa egy kódszekcióra:

;{Fõprogram
ORG $3000       ;A program kezdõcíme
LD A,$45
;}

A { karakter után írt szöveg a szekció neve, mely összecsukás után látható:

 

 A programon belül minden ablak tetszõleges helyre helyezhetõ, vagy dokkolható. Az ablakok helyzetét a TVCStudio újraindítás esetén is megjegyzi.

 

A program által támogatott gyorsbillentyûk

CTRL+S: program mentése

CTRL+O: program megnyitása

CTRL+K: kód igazítása (jelenleg csak assembly program esetén támogatott)

F9: Program fordítása

CTRL+N+A: új assembly program létrehozása

CTRL+N+B: új BASIC program létrehozása

CTRL+F: keresés

F12: Ugrás a címke definícióhoz

CTRL+Plusz: ZoomIn

CTRL+Minus:ZoomOut

CTRL+Balra: Navigálás a szerkesztõben szavanként balra

CTRL+Jobbra: Navigálás a szerkesztõben szavanként jobbra
